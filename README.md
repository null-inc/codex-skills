# Codex Skills

A collection of reusable Skills for Codex CLI.

These Skills are designed to improve common developer workflows by providing structured, reusable prompts for tasks such as project discovery, conversation summarization and project handoffs.

## Available Skills

### `/discovery`

Interview the user to refine a project idea before producing a project plan.

### `/transfer`

Transfer the current project state so another agent can continue working with minimal additional context.
